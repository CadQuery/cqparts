class Applicator(evaluator, selector)¶

Bases: object

The applicator performs 2 roles to apply a fastener to workpieces

  1. Translate & rotate given fastener
  2. Change workpieces to suit a given fastener

Translation is done first because the fastener is sometimes used as a cutting tool to subtract from mating part(s) (eg: thread tapping).

Male fastener part; with an external thread

                           _________  __ head height
                          | \/   \/ |
        z=0 __ _________  |_/\___/\_| __ z=0 (on x/y plane, +z is up)
               \       /    |     |
head height __  \     /     |     |
                 |   |      |     |   __ neck length (excludes taper)
                -\---/-     -\---/-
                -|---|-     -|---|-
                -|---|-     -|---|-
                -|---|-     -|---|-   __ tip (length from bottom)
                -\---/-     -\---/-
                  \_/         \_/     __ length

Warning

Tip thread tapering has not been implemented, except in the simplified model.
