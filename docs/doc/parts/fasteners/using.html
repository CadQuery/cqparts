Using Fasteners¶

The cqparts.fasteners collection has some utilities to automatically apply and select an apropriate fastener. What they do, and how to use them is documented below.

Using these tools is not mandatory; you’ll always have the option to explicitly select your fastener with the placement and orientation of your choosing.

Evaluation¶

An Evaluator instance is an assessment of the workpieces intended to be fastened along a given vector.

An Evaluator instance is required for a Selector, and an Applicator.

If a fastener Selector is used to find an appropriate fastener, that same Evaluator instance can be passed to the Applicator.

Evaluations are given, the what, and the where:

  • what: list of parts (those that may be effected, should be as short as possible)
  • where: linear edge, a vector along which the fastener is intended to be applied

A successful evaluation will yield:

  • list of parts effected (in order)
  • start & finish points on each effected part (the start being closer to the head)

There are 2 methods of evaluation

  • vector : faster, but may miss something
  • cylindrical : slower, but will find all interfacing solids
